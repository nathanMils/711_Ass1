# Assignment 2 Hyperparameter Tuning and Hybrid Learning

## How To Run

### Requirements:
- Docker Desktop with WSL2
- Docker Compose plugin

```bash
git clone https://github.com/nathanMils/711_Ass1.git

cd 711_Ass1

docker compose up
```
